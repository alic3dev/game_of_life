# game_of_life

an implementation of the game of life utilizing [`cexil`](https://github.com/alic3dev/cexil)

<img width="1377" alt="game_of_life" src="https://github.com/user-attachments/assets/29e4873d-3a05-4bb6-8392-3f92534531cb" />

## running

```sh
./output/game_of_life
```

## development

### prerequisites

- [`cexil`](https://github.com/alic3dev/cexil)
- [`interrupt_handler`](https://github.com/alic3dev/interrupt_handler)

#### environment_variables

- `CEXIL_DIRECTORY`
- `INTERRUPT_HANDLER_DIRECTORY`

### building

```sh
make
```

### cleaning

```sh
make clean
```
